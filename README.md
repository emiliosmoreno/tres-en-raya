# Tres en Raya

Juego "Tres en Raya" también conocido cómo "Gato" el cual
se juega de manera autonoma,
Construido con ayuda de Python y Pygame.

## Requerimientos:

Python >= 2.7

Pygame >= 1.9.1

## Ejecutar:

    python ./main.py
